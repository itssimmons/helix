Helix Server - A lightweight and high-performance fully extensible and minimalist web server framework for Bun.js.

## Features

- **Lightweight & Fast**: Built on Bun.js for optimal performance.
- **Extensible**: Easily add middleware and plugins to enhance functionality.
- **Minimalist**: Simple and clean API for quick development.
- **TypeScript Support**: Fully typed for better developer experience.
- **Routing**: Flexible routing system to handle various HTTP methods and routes.
- **Middleware Support**: Integrate custom middleware for request processing.
- **Open Source**: Community-driven with contributions welcome.
