Esta guía te ayudará a crear tu primera aplicación de servidor Helix.

## Pré-requisitos

- [Bun](https://bun.sh) instalado en tu máquina.

## Instalación

Primero, instala el paquete del servidor Helix:

```bash
bun add @helix-server/core
```

## Crea Tu Primer Servidor

Crea un nuevo archivo llamado `index.ts`:

```bash
touch index.ts
```

Agrega el siguiente código a `index.ts`:

```typescript
import helix from '@helix-server/core';

helix.get({
	route: '/',
	async handler(ctx) {
		ctx.res.setBody('hello world');
	},
});

helix.run(3000, '127.0.0.1');
```

## Ejecuta Tu Servidor

Inicia el servidor con:

```bash
bun run index.ts
```

Deberías ver el mensaje:

```
Helix server is running at http://127.0.0.1:3000
```

Visita [http://127.0.0.1:3000](http://127.0.0.1:3000) en tu navegador para ver "hello world".
